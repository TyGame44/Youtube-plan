# They put a fly in the matrix



In 2026, Flygym made headlines. Flygym is a reinforcement learning environment for simulating the adult fruit fly.



Critics of this publication quickly pointed out that Flygym is only the physical embodiment of a virtual fly. It is only able to simulate fly movement, touch, sight, and smell.



What most of these critics missed is that fly-gym ships with an example that uses fly-vis. This is a neuro-anatomically accurate simulation of the flies visual processing.



Advancements from fly-vis have shown that you only need to know two details about the neuro-anatomy of the brain to simulate it. You only need to know the synapse connections and the polarity of each neuron. Fly-vis also demonstrated that machine learning can be used to predict resting voltages and action potentials using only the synapse and polarity data.



Fly vis also showed advancements in simulation by discovering new groupings of neurons and minimizing the types of neurons in the fly's visual processing.



In addition to this, companies like Eon have reproduced the full fly brain in simulation and are finding that the predicted neural activity matches a real biological fly with 91% accuracy.



Eon systems took this current progress and in 2024 they showed that the leaky integrate and fire model of the neuron was a strong enough predictor for their 91% accuracy result.



Eon's mission statement is to turn human brain simulation into a tech sprint rather than a century long research project. I believe in them. With better brain simulation algorithms, advancements in GPU processing, and more models for the open source community to collaborate on, we are closing in on the possibility of a digital immortality.



To address the main complaint of the brain's complexity vs a computer, I disagree. I believe that it is a matter of an optimization problem for different media. If you try to start with cpu, it seems impossible. If you approach it with a gpu, it seems possible but overly complex. If you start to use neural-specific hardware you are already much closer. See China's Darwin Monkey chip that can simulate 100 billion synapses. Not quite a human’s quadrillion synapses but it is getting closer. Advancements in quantum computing or even diamond integrated circuits may already extend Moore's law into this range.



If that doesn’t work there is potential for a paradigm shift in computing. A new gpu per se. Extropic is developing “Thermodynamic computation”. This could improve the entire leaky integrate and fire model to the point of having it run on one thermal compute unit per neuron.
