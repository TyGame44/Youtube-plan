# How close are we to solving P versus NP

A recent discovery and math shows that every operation can be represented with the EML operator.

This looks complicated because the function is E to the X minus the natural log of Y.

This overcomplicates math so why would it be useful?

This goes back to our fundamental understanding of computer science.

Alan Turing and Alonso Church described turing machines and lambda calculus.

If we can reduce everything in math to one function, we can convert that function into a lambda diagram.

This lets us recursively optimize it.

It unlocks one of the most powerful optimization tools in computer science and lets us apply it to every existing mathematical identity.

Now I just wonder if we can further simplify EML and/or how we can better convert EML optimizations into byte code.
